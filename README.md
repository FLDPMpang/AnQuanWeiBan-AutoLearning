# 说明

麦课/安全微伴刷课工具

> 本项目 Fork 自[该项目](https://github.com/WeiYuanStudio/AutoWeiBan)

## 使用方法

1. 首先需要在运行设备上安装 Python3 运行环境和 requests 库
   `pip intall requests`

2. 使用 python 自带的 IDLE 编辑运行目录下的 main.py 或在命令行下运行
   `python3 main.py`

3. 依据脚本提示打开网址扫描二维码登录即可

## 使用时请在 main.py 中将院校码修改为自己学校的院校码
